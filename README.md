# MorphologyDocumenter

A one-page web-app for documenting the morphology and lexicography of a Greek word in a specific literary context.

## How to use
1. Open `index.html` in any modern browser (no server required).
2. [Live demo](https://eumaeus.github.io/MorphologyDocumenter/) (GitHub Pages)
3. Place `lsj_urns.txt` in the same folder for LSJ URN validation.

## Features
- BetaCode ↔ Unicode conversion (with terminal sigma)
- Intelligent POS tag builder (dynamic menu enabling)
- Click-to-edit entries
- Two-file TSV export for Dramaturg.jl
